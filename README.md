# myntra-website-clone
This project was initially developed using HTML and CSS to replicate the visual design of the Myntra website. It has now been enhanced with JavaScript to add interactivity and improve the overall user experience.

**Features**
- Navigation bar with categories
- Add-to-Bag Functionality
- Product listing section
- Shopping bag icon with item count
- Real-time bag item counter using JavaScript
- Data storage using Local Storage
- Interactive user interface

**Technologies Used**
- HTML
- CSS
- JavaScript

**How It Works**
1. Browse the available products displayed on the homepage.
2. Click the **Add to Bag** button to add products to your shopping bag.
3. The bag count updates instantly using JavaScript whenever an item is added.
4. Selected items are stored in the browser using Local Storage.
5. Clicking the **Bag** icon opens the shopping bag page.
6. The bag page displays all selected products along with their details and pricing.
7. JavaScript calculates and displays the **total price** of all items in the bag.
8. Users can remove items from the bag, and the total amount updates automatically.
9. The data remains available even after refreshing the page, ensuring a seamless shopping experience.

**Project Purpose**

The purpose of this project is to recreate the core shopping experience of the Myntra website while strengthening frontend development skills. Initially built using HTML and CSS for the user interface, the project was later enhanced with JavaScript to add dynamic functionality and interactivity.
