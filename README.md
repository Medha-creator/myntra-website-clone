# myntra-website-clone
This project is a basic clone of the Myntra website built with HTML and CSS. It focuses on recreating the look and feel of the original website.

**Features**
- Homepage design
- Product display layout
- Navigation bar
- Category sections
- Footer section with useful information

**Technologies Used**
- HTML
- CSS

**Project Purpose**

This project was made to enhance my HTML and CSS skills by building a basic e-commerce website layout.
